# MB Classical Bot
As a long-time editor of the <a href="http://www.musicbrainz.org">MusicBrainz</a> database, I have
come across several types of edits that could be easily automated. My primary focus is on classical
music, so this bot will be designed to make edits only to entities that are related to this genre.
All edits are intended to follow <a href="https://musicbrainz.org/doc/Style/Classical">Classical
Style Guidelines</a>.

The number of edits is kept small at first, and even though it runs daily, I'm planning to run 
it only when I'm able to vote on all edits created by it.
